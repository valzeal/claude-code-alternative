## Usage

### Command Line Interface

The CLI provides 7 main commands:

```bash
# Analyze code
cca analyze script.py --language python --output json

# Generate code from prompt
cca generate "Create a function to sort an array" --language javascript

# Review code quality
cca review app.js --language javascript --output text

# Debug code issues
cca debug server.py --language python

# Generate documentation
cca docs main.py --language python

# Fix code issues
cca fix utils.py --language python

# Manage configuration
cca config --list
cca config --set api_key=your_key
```

### Configuration

Configuration is stored in `~/.cca/config.json`:

```json
{
  "api_key": "your_api_key_here",
  "server_url": "http://localhost:8000",
  "cache_enabled": true,
  "default_language": "python",
  "default_output_format": "text"
}
```
